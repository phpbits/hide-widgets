Add options to hide specific widgets on desktop and/or mobile & iPad screen sizes.

== Description ==

This plugin will create additional field options on your widgets. Check Screenshot section for more details. Thanks!


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. That's it :) , additional widget options will be added
